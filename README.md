# oh_my_zsh_themes
Just an another oh my zsh theme


``` 
cd ~/.oh-my-zsh/custom/theme/
git clone https://github.com/gauthamarcot/oh_my_zsh_themes.git
mv oh_my_zsh_themes/* .
```
# set the theme like this in oh-my-zsh
  ```
  vim/nano/gedit .zshrc
  ```

  ZSH_THEME="arcot/hacker"
### Note:
only works in linux
